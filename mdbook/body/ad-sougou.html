4.1 简介

可以将一些常用的 Markdown 短语定义成搜狗短语，以便提高输入效率。借助搜狗输入法自带的 【自定义短语配置】 功能，我们可以将一些常用的 Markdown 格式语句定义成搜狗短语，以提高输入效率。只要输入几个字母就出现一大串代码。

4.2 定义方法

你可以在百度或知乎上搜索关键词【搜狗自定义短语】，相关教程很多，例如：输入法“自定义短语设置”还可以这么玩；搜狗输入法-自定义短语设置的神奇妙用。这里，我只做简要说明。

4.2.1 基本规则

在搜狗输入法中敲入 whn 三个字母后，会弹出如下备选词组：

  • 短语：就是你打字时的缩写，通常是个简单易记的英文缩写，如 whn；
  • #：表示输入法中显示的序号 (如上图中，若依次在键盘上敲入 whn → 2，则会输入 我和您。)

按此逻辑，如果我们预先定义如下规则：

pf,1 = [-PDF-]()

则当你采用「搜狗输入法」打字时，只需输入 lj，屏幕上第一个显示出来的词条就是 [-PDF-]()。

4.2.2 自定义短语配置方法

依次点击 [属性设置 → 高级 → 自定义短语设置 → 直接编辑配置文件]，然后在打开的文本文件中添加你的配置即可。

Tip

  • 对于 Windows 用户而言，可以按 Win 键 (Ctrl 右侧的键)，在弹出程序列表中找到 搜狗输入法，然后依次点击 [设置 → 高级 → 自定义短语设置 → ……]

  • 对于 MacOS 用户，可以点击屏幕右上角的 输入法图标，选择 偏好设置，然后依次点击 [高级 → 自定义短语设置 → 直接编辑配置文件] 进行修改。
