2  Markdown 基本语法

https://quarto.org/docs/authoring/markdown-basics.html

2.1 标题

Markdown 使用 # 符号表示标题。# 的数量代表标题的级别，从 1 级到 6 级。

# 一级标题
## 二级标题
## 三级标题
……
###### 六级标题

2.2 段落与换行

  • 段落：段落之间需要空一行。
  • 段内换行：在行尾添加两个空格，或者使用 <br> 标签。
这是第一段。

这是第二段。  
这是同一段的换行。

需要说明的是，使用 Markdown 撰写的文档无需在在段首空两格。若需添加，可以采用 HTML 标记符 &emsp; (参见 ?sec-html-syntax)：

&emsp;&emsp;多数情况下，md 文档主要通过网页形式发布。  
因此，首行无需空两个字符。 …………………

  多数情况下，md 文档主要通过网页形式发布。
因此，首行无需空两个字符。 …………………

2.3 列表

Markdown 支持有序列表和无序列表。

Tip
  • 建议在正文文字与列表之间添加一个空行，以确保在多数编辑器中都能正常渲染。Quarto (参见 ?sec-quarto-intro) 对此有明确的格式要求。
  • 若在列表项之间加入空行 (仅需一个)，渲染后的列表行间距会增大，视觉效果更清晰；反之，生成的列表比较紧凑。

2.3.1 无序列表

使用 -、* 或 + 表示无序列表。二级列表首行缩进两个字符，以此类推。

下面是一个二级列表：

- 回归分析
- 机器学习
  - 随机森林

下面是一个二级列表：

  • 回归分析
  • 机器学习
    • 随机森林

2.3.2 有序列表

使用数字加点表示有序列表。

1. 第一项
2. 第二项
   1. 子项 1
   1. 子项 2
3. 第三项
   - 子列表
   - 子列表
Note
  1. 有序列表前的序号不必严格遵守 1. 2. 的顺序，可以写为 1. 1.，甚至是 1. 3. 2.
  2. 二级子项或子列表前要保留三个空格。
  1. 第一项
  2. 第二项
    1. 子项 1
    2. 子项 2
  3. 第三项
    • 子列表
    • 子列表

2.4 强调

使用 * 或 _ 表示斜体，** 或 __ 表示粗体。

*斜体* 或 _斜体_  
**粗体** 或 __粗体__  
***粗斜体*** 或 ___粗斜体___

斜体 或 斜体
粗体 或 粗体
粗斜体 或 粗斜体

Note
  1. 用 * 或 ** 包围文字时，文字两侧不要加空格，否则在有些编辑器中无法渲染。
  2. 在有些编辑器中，加粗符号两侧也要加空格，即 请 **加粗** 这些内容。

2.5 链接与图片

2.5.1 链接

使用 [文本](URL) 表示链接。

[Google](https://www.google.com)

2.5.2 图片

使用 ![替代文本](图片URL) 插入图片。如果是本地图片，可以使用相对路径。

![Markdown Logo](images/markdown-logo.png)
Note
  • 图片的文件名中不要包含空格和特殊字符。
  • 如果插入的是本地图片，请务必使用相对路径。
  • 设置图片尺寸和位置可以使用 HTML 标签：
<img style="width: 550px" src="images/markdown-logo.png">

2.6.1 行内代码

使用 ` 包裹行内代码。

这是 `行内代码` 示例。

2.6.2 代码块

使用 包裹代码块，并指定语言（可选）。

2.7 引用

使用 > 表示引用。

> 这是一个引用。
> 引用可以有多行。  
> 注意：多行引用时，每行末尾需加两个空格。

2.8 表格

使用 | 和 - 创建表格。

| 左对齐 | 居中 | Default | 右对齐 |
|:------|:----:|---------|-------:|
|  0.12 |  12  |    12   |    1.23|
|  1.2  |  123 |   123   |   12.31|
左对齐 居中 Default 右对齐
0.12 12 12 1.23
1.2 123 123 12.31

合并单元格： Markdown 本身不支持合并单元格，但可以通过 HTML 实现。参考 HTML 表格教程。

2.9 分割线

使用 -、* 或 _ 创建分割线（至少三个符号）。

---

2.10 注释

Markdown 本身不支持注释，但可以通过 HTML 注释实现。

<!-- 这是一个注释，不会显示在渲染结果中 -->

2.11 转义字符

如果需要显示特殊字符（如 #、* 等），可以使用反斜杠 \ 进行转义。

这是 \*不是\* 斜体。

常见转义字符： - \#：# - \*：* - \_：_ - \\：
- \``： - \{：{ - \}：}

2.12 数学公式

Markdown 支持 LaTeX 语法插入数学公式。使用 $ 包裹行内公式，$$ 包裹独立公式。

行内公式：$E = mc^2$

独立公式：
$$
\int_{a}^{b} x^2 dx
$$

2.13 扩展语法

Markdown 有许多扩展语法（如 GitHub Flavored Markdown），支持更多功能： - 任务列表 - 内嵌 HTML - 脚注 - 数学公式

我们将在后续章节中详细介绍这些扩展语法。
